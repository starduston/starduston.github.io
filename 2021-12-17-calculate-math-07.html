矩阵与数值分析笔记 Part07

Posted on 2021-12-17 In Note

Abstract

这是矩阵与数值计算第七章的笔记

7.1 线性单步法

7.1.1 一阶常微分方程的初值问题

对常微分方程：

\[\begin{cases} u' = f(t,u) & ,a < t < b\\ u(a) = u_0 \end{cases}\]

常用的线性单步法：

Euler法

\[u_{n+1} = u_n + hf(t_n, u_n)\]

隐式Euler公式

\[u_{n+1} = u_n + hf(t_{n+1}, u_{n+1})\]

梯形公式

\[u_{n+1} = u_n + \frac{h}{2}[f(t_n, u_n) + f(t_{n+1},u_{n+1})]\]

7.2 线性多步法

7.2.1 线性多步法

一般公式

\[\sum_{j=0}^k \alpha_j u_{n+j} = h \sum_{j=0}^k \beta_j f_{n+j} , \alpha_k \neq 0\]

待定系数法构造一般线性二步法公式

对于一般线性二步法公式：

\[\alpha_2 u_{n+2} + \alpha_1 u_{n+1} + \alpha_0 u_{n} = h(\beta_2 f_{n+2} + \beta_1 f_{n+1} + \beta_0 f_n)\]

计算其\(c_p\)：

\[c_0 = \alpha_0 + \alpha_1 + \alpha_2\]

\[c_1 = (\alpha_1 + 2\alpha_2) - (\beta_0 + \beta_1 + \beta_2)\]

\[c_p = \frac{1}{p!}(\alpha_1 + 2^p \alpha_2) - \frac{1}{(p-1)!}(\beta_1 + 2^{p-1} \beta_2)\]

根据其阶数令\(\alpha_p = 0\)，求出相应的系数（可以令\(\alpha_2=0\)，\(\alpha_0=a\)设为待定量，然后求出其他系数）。

若\(c_p=0, c_{p+1} \neq 0\)，则阶数为\(p\)（二步法最高阶数为4），局部截断误差为：

\[R_{n+1}(h) = c_{p+1} h^{p+1} u^{(p+1)}(t) + O(h^{p+2})\]

特征多项式

  • 第一特征多项式

\[\rho(\lambda) = \sum_{j=0}^k \alpha_j \lambda^j\]

  • 第二特征多项式

\[\sigma(\lambda) = \sum_{j=0}^k \beta_j \lambda^j\]

判断截断误差阶的方法

计算\(\rho(\lambda)-ln(\lambda)\sigma(\lambda) = \rho(z+1)-ln(z+1)\sigma(z+1)\)，\(ln(z+1)=\sum_{k=1}^{\infty}\frac{(-1)^{n-1}}{n}z^n\)次数最低的项，阶段误差阶为其次数减一，相应的系数为局部截断误差主项系数

收敛性

线性多步法收敛需满足两个条件：

  • 根条件：第一特征多项式\(\rho(\lambda)\)的的根均落在单位圆内或单位圆上（\(|\lambda| \le 1\)），且落在圆上的根均为单根

  • 线性多步法至少为一阶

绝对稳定区间的计算

令\(f_n = \mu u_n\)，得到差分方程：

\[\sum_{j=0}^k \alpha_j u_{n+j} = \mu h \sum_{j=0}^n \beta_j u_{n+j}\]

从而特征方程为:

\[\rho(\lambda) - \bar{h} \sigma(\lambda) = 0,(\bar{h}=\mu h)\]

要满足绝对稳定，需使得\(|\lambda| \lt 1\)，由此解出\(\bar{h}\)的范围即为绝对稳定区间。

  • 实系数二次方程\(\lambda^2 - b\lambda - c = 0\)的根再单位圆的充要条件为\(|b| < 1 - c < 2\)
